# Stata code

This section aims to cover the Stata estimation commands from various packages.


All the code uses the following set of symbols:

| Symbol | Description | 
| $$ i $$ | panel id |
| $$ t $$ | time variable |
| $$ Y $$ | outcome variable |
| $$ D $$ | =1 if treated observation |
| $$ \epsilon $$ | error term |
